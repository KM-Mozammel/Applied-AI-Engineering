এখন একটি গুরুত্বপূর্ণ প্রশ্ন আসে: "NumPy যদি এত শক্তিশালী হয়, তাহলে Pandas কেন শিখব?"

উত্তর: NumPy মূলত সংখ্যা (Numerical Data) নিয়ে কাজ করার জন্য তৈরি।

কিন্তু বাস্তব Machine Learning Dataset কেমন হয়?

Name	Age	City	Salary
Rahim	25	Dhaka	50000
Karim	30	Chattogram	65000

এখানে String, Number, Missing Value—সব একসাথে আছে। এ ধরনের Tabular Data নিয়ে কাজ করার জন্যই Pandas তৈরি হয়েছে।
