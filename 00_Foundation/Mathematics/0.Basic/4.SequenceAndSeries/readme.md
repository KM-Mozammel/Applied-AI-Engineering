1, 2, 3, 4, 5, ...

# Each number is called a term.
a₁ = 1
a₂ = 2
a₃ = 3
...

Day 1 → 100
Day 2 → 200
Day 3 → 300
Day 4 → 400

# Sequence Analogy
Imagine a train: [1]--[2]--[3]--[4]--[5]
Each compartment is a term.
The whole train is the sequence.

# Sequence -------->
When you see: 1, 2, 3, 4, 5
Ask: Are the numbers just listed?
Yes → Sequence

# Sequence = Numbers


When you see: 1 + 2 + 3 + 4 + 5
Ask: Are the numbers being added?
Yes → Series
