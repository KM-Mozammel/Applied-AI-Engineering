# In AI Engineering:
A word embedding is actually a vector.

Example:

cat = [2, 3]
dog = [4, 5]

To compare vectors we measure the angle between them.

Cosine similarity:

cos(θ)

If:

cos(θ) = 1

Vectors point in same direction.

If:

cos(θ) = 0

Vectors are unrelated.

If:

cos(θ) = -1

Vectors point opposite directions.

This is used in:

Search engines
RAG systems
Embeddings
Recommendation systems
LLMs
