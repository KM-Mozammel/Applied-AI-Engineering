Calculus শিখেছো "Change" বোঝার জন্য। Probability শিখছো "Uncertainty" বোঝার জন্য। এখন Statistics শিখবে "Data থেকে জ্ঞান (Knowledge) বের করার বিজ্ঞান" হিসেবে।

# ইতিহাস: প্রাচীন রাজারা জানতে চাইতেন—

"আমার রাজ্যে কত মানুষ আছে?"
"কত সৈন্য আছে?"
"কত কর (Tax) আদায় হবে?"

এই তথ্য সংগ্রহ করার কাজ থেকেই Statistics-এর জন্ম।

"Statistics" শব্দটি এসেছে ল্যাটিন Status (State) থেকে, কারণ প্রথমে এটি রাষ্ট্র পরিচালনার জন্য ব্যবহৃত হতো। কিন্তু ১৮শ–১৯শ শতাব্দীতে বিজ্ঞানীরা বুঝলেন, "Data-এর ভিতরেও গল্প লুকিয়ে থাকে।" সেই গল্প বের করার বিজ্ঞানই Statistics।

| Concept                 | মূল প্রশ্ন                                                        |
| ----------------------- | ----------------------------------------------------------------- |
| **Mean**                | ডেটার গড় কত?                                                     |
| **Median**              | মাঝের মান কী?                                                     |
| **Mode**                | সবচেয়ে বেশি কোন মানটি এসেছে?                                     |
| **Variance**            | ডেটা কতটা ছড়িয়ে আছে?                                            |
| **Standard Deviation**  | গড় থেকে সাধারণত কত দূরে ডেটা থাকে?                               |
| **Covariance**          | দুটি ভেরিয়েবল কি একসাথে পরিবর্তিত হয়?                           |
| **Correlation**         | সেই সম্পর্ক কতটা শক্তিশালী?                                       |
| **Confidence Interval** | Sample থেকে Population সম্পর্কে কতটা নির্ভরযোগ্য অনুমান করা যায়? |
| **Hypothesis Testing**  | দেখা পার্থক্যটি কি সত্যি, নাকি কাকতালীয়?                         |
| **P-Value**             | প্রমাণ কতটা শক্তিশালী যে Null Hypothesis বাতিল করা উচিত?          |
