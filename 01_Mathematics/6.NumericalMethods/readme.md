# Numerical Methods (জন্ম):
ইতিহাস: প্রাচীনকালে গণিতবিদরা কাগজ-কলমে সমীকরণ সমাধান করতেন। কিন্তু বিজ্ঞান যত এগোতে লাগল, তত জটিল সমস্যা সামনে এল। যেমন—

পৃথিবীর আবহাওয়া পূর্বাভাস
মহাকাশযানের কক্ষপথ
Fluid Simulation
Neural Network Training
